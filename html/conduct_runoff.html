conduct_runoff {rcvr}R Documentation

Conduct instant runoff

Description

Conducts a ranked choice instant runoff. Does not resolve ties, as yet.

Usage

conduct_runoff(
  cntst,
  sequentialU = FALSE,
  max_U = 2,
  sequentialO = FALSE,
  max_O = 1,
  june2018 = FALSE,
  quietly = TRUE
)

Arguments

cntst

A contest instance.

june2018

If TRUE duplicates official State tabulations.

quietly

When FALSE many routines get noisy.

Value

A tabulation instance.
