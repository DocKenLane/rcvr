read_contest_keys {rcvr}R Documentation

Read contest keys

Description

Get names and such for loading election data.

Usage

read_contest_keys(contest_str, path = getwd())

Arguments

contest_str

Must reference an object value in key_values.txt.

path

Some folder, somewhere. Typically defaults to working directory.

Value

Names for an election

Examples

read_key_values(contest_str = "d2_June", path = "My data folder")
