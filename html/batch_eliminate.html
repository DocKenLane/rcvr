batch_eliminate {rcvr}R Documentation

Eliminate alternatives

Description

Eliminates alternatives by subsetting continuing_alternatives.

Usage

batch_eliminate(
  round,
  alternative_code,
  continuing_alternatives,
  quietly = TRUE
)

Arguments

round

An instance of a RC/IR runoff round.

alternative_code

Character vector of codes "[A-N]" for representing alternatives in a contest.

continuing_alternatives

A subset of alternative_code representing continuing alternatives in a tabulation.

quietly

When FALSE many routines get noisy.

Value

A character vector of subset of continuing_alternatives.
