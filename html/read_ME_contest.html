read_ME_contest {rcvr}R Documentation

Read Maine contest files

Description

Used to read Cast Vote Record files (*.xlsx) for a State of Maine Ranked Choice Instant Runoff election.

Usage

read_ME_contest(source, CVR_files, original = FALSE, quietly = TRUE)

Arguments

source

Folder of CVR files.

CVR_files

Cast Vote Record file names.

original

If original == TRUE, contest_str must match an object value in contest_keys.txt file in working directory, with cvr_url value and all cvr_file values needed to reconstruct the election.

quietly

When FALSE many routines get noisy.

Value

A list of dataframes.
